% 3. Summer Training/ Internship Project Detail
\chapter{Summer Training/ Internship Project Detail}

\section{Problem Statement}
Managing a large fleet of vehicles is complex, requiring real-time tracking, maintenance scheduling, and expense management. Without an automated system, the overhead increases and the operational efficiency drops significantly.

\section{Project Outcomes}
Developed the FleetMaster application which allows admins to efficiently manage vehicles, drivers, expenses, and maintenance logs. The platform includes a secure backend API and a responsive React frontend dashboard featuring role-based access for mechanics and admins.

\section{Technologies used}
Frontend: React, Vite, Tailwind CSS, React Router\\
Backend: Node.js, Express.js, Mongoose\\
Database: MongoDB\\
Authentication: JSON Web Tokens (JWT)
